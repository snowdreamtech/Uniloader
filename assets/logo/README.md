# Uniloader Brand Assets

This directory contains the official logo for **Uniloader**.

## Formats and Sizes

The logo is provided in both **PNG** (transparent background, lossless) and **JPEG** formats in the following dimensions:

- **1024x1024** (Original High-Res)
- **512x512** (Standard)
- **256x256** (Medium)
- **128x128** (Small / Avatar)
- **64x64** (Icon)
- **32x32** (Favicon)

### Example Usage

```markdown
<!-- Standard Markdown Logo Include -->
<p align="center">
  <img src="logo-512x512.png" alt="Uniloader Logo" width="250"/>
</p>
```

## Design Concept

The Uniloader logo features a modern 3D glowing neon box (representing the loader) receiving dynamic, multi-colored data streams and blocks (representing various operating systems and software packages). It symbolizes Uniloader's role as a _Universal_, cross-platform software orchestration engine.
